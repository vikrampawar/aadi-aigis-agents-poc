# Aigis Agents — Project Instructions

## Mermaid Diagrams

When creating or editing Mermaid diagrams in this repo, use the `/mermaid-theme` skill to apply the standard dark theme. All diagrams must follow that theme's colour palette and init directives.

## Documentation Structure

- Business specification: `docs/SPECIFICATION.md`
- Excalidraw diagrams: `docs/diagrams/NN-short-description.excalidraw`
- Diagrams use dark theme (dark background, light text) — see `docs/README.md` for conventions
- When adding a new diagram, number it sequentially and add a reference in the spec
